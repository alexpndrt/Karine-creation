# 01 - Contexte du Projet

## Informations générales

- **Nom du projet** : Karine Creation
- **Type de projet** : Site vitrine artisanal pour activité de couture sur mesure
- **Client** : Karine (couturière professionnelle indépendante)
- **Chef de projet** : Alex Pondart (développeur CDA - fils de Karine)
- **Date de création** : Janvier 2026
- **Échéance souhaitée** : Avril 2026 (3-4 mois de développement)

## Contexte métier

Karine est une couturière professionnelle avec plus de 15 ans d'expérience dans la création de vêtements et d'accessoires sur mesure. Son activité comprend :

- **Créations sur mesure** : Robes de mariée, costumes homme, tenues de cérémonie
- **Retouches et modifications** : Ajustements, réparations, transformations
- **Conseils stylistiques** : Accompagnement personnalisé pour les choix de tissus et styles
- **Service de proximité** : Travail artisanal de qualité dans la région

Le secteur de la couture sur mesure est en pleine évolution avec une demande croissante pour des pièces uniques et durables, en opposition à la fast fashion.

## Objectifs du site web

### Objectif principal

**Valoriser l'expertise artisanale de Karine et faciliter la prise de contact avec de nouveaux clients**

### Objectifs secondaires

- **Présenter le savoir-faire** : Montrer la diversité des créations à travers une galerie photo professionnelle
- **Générer des leads qualifiés** : Formulaire de contact optimisé pour demandes de devis
- **Établir la crédibilité** : Témoignages clients, parcours professionnel, certifications
- **Éduquer les clients** : Expliquer les processus de création et les valeurs de l'artisanat

## Valeurs ajoutées attendues

### Visibilité

- **Référencement local** : Apparaître en première position pour "couture sur mesure [ville]"
- **Présence digitale** : Compléter les réseaux sociaux existants (Instagram, Facebook)
- **Rayonnement régional** : Attirer des clients au-delà du bouche-à-oreille traditionnel

### Crédibilité

- **Image professionnelle** : Site moderne reflétant la qualité du travail
- **Preuves sociales** : Portfolio détaillé, témoignages clients vérifiés
- **Transparence** : Tarifs clairs, processus expliqué, délais communiqués

### Conversion

- **Génération de leads** : Formulaire de contact optimisé UX
- **Facilitation RDV** : Processus simple de prise de contact
- **Fidélisation** : Newsletter pour maintenir le lien avec les clients

### Relationnel

- **Proximité humaine** : Reflet de l'approche artisanale et personnalisée
- **Confiance** : Transparence sur les méthodes de travail et valeurs
- **Accompagnement** : Guide pour aider les clients dans leurs choix

## Contraintes identifiées

### Techniques

- **Technologies libres** : Next.js + TypeScript recommandé pour performance et SEO
- **Responsive obligatoire** : Mobile-first pour l'expérience utilisateur
- **Performance** : Site rapide et accessible sur tous supports
- **Sécurité** : Conformité RGPD et protection des données clients

### Budgétaires

- **Budget cible** : 8 000 - 12 000 € TTC pour développement complet
- **Priorité coût/efficacité** : Solutions open-source privilégiées
- **ROI attendu** : Récupération investissement sous 12 mois
- **Maintenance** : Budget annuel ~500-800€ pour hosting et mises à jour

### Temporelles

- **Délai souhaité** : 3-4 mois pour livraison complète
- **Planning flexible** : Possibilité ajustements selon contraintes personnelles
- **Itérations** : Développement itératif avec validations intermédiaires
- **Formation** : Temps dédié à l'apprentissage des technologies

### Réglementaires

- **RGPD** : Conformité collecte et traitement données personnelles
- **RGAA niveau AA** : Accessibilité pour tous les utilisateurs
- **Mentions légales** : Informations juridiques obligatoires
- **Droits d'auteur** : Respect propriété intellectuelle des photos
