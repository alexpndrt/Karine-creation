# 04 - Gestion des Contenus

## Stratégie de contenu

### Objectifs
- **Valoriser l'expertise** : Montrer le savoir-faire artisanal
- **Inspirer confiance** : Témoignages et processus transparents
- **Faciliter conversion** : Guides pratiques et informations claires
- **Optimiser SEO** : Contenu riche et structuré

---

## Inventaire des contenus

### Contenus textuels

#### Page Accueil
- **Hero** : Accroche impactante (150 caractères)
- **Présentation** : Qui est Karine (300 mots)
- **Services phares** : 3 services principaux (50 mots chacun)
- **CTA** : Appel à l'action clair

#### Page Portfolio
- **Introduction** : Présentation de l'approche (200 mots)
- **Catégories** : Descriptions par type de création (100 mots chacune)
- **Légendes photos** : Matières, techniques, contexte (50 mots par photo)

#### Page Services
- **Service 1-3** : Description détaillée (200-300 mots chacun)
  - Processus de travail
  - Délais indicatifs
  - Garanties
  - Exemples concrets

#### Page Tarifs
- **Grille tarifaire** : Prix clairs et justifiés
- **Forfaits** : Offres groupées avantageuses
- **Conditions** : Modalités et paiement

#### Page Contact
- **Coordonnées** : Adresse, téléphone, email
- **Horaires** : Jours et créneaux d'ouverture
- **Formulaire** : Champs optimisés UX

#### Page À propos
- **Parcours** : Histoire professionnelle (400 mots)
- **Valeurs** : Philosophie de travail (200 mots)
- **Atelier** : Présentation de l'espace de travail
- **Équipement** : Outils et machines utilisés

### Contenus visuels

#### Photos portfolio (priorité haute)
- **Minimum** : 20 photos haute qualité (3000x2000px minimum)
- **Formats** : JPG/PNG, optimisation WebP
- **Catégories** :
  - Robes de mariée (5 photos)
  - Costumes homme (4 photos)
  - Retouches (6 photos)
  - Créations originales (5 photos)

#### Photos atelier/équipe (priorité moyenne)
- **Atelier** : 3-5 photos de l'espace de travail
- **Équipement** : Photos des machines/outils
- **Ambiance** : Photos lifestyle de l'atelier

#### Éléments graphiques (priorité basse)
- **Logo** : Version vectorielle + déclinaisons
- **Icônes** : Services, contact, navigation
- **Illustrations** : Personnalisation possible

### Contenus techniques

#### Métadonnées SEO
- **Titles** : 50-60 caractères optimisés
- **Descriptions** : 150-160 caractères accrocheurs
- **Mots-clés** : couture, retouches, sur-mesure, [ville]

#### Données structurées (Schema.org)
- **Organisation** : Coordonnées et horaires
- **Services** : Descriptions et tarifs
- **Portfolio** : Galerie avec métadonnées

## Plan de contenu

### Phase 1 (MVP)
- **Textes essentiels** : Accueil, Services, Contact
- **Portfolio minimum** : 10 photos représentatives
- **Informations de base** : Coordonnées, horaires

### Phase 2 (Enrichissement)
- **Contenu détaillé** : À propos complet, blog
- **Portfolio étendu** : 30+ photos organisées
- **Témoignages** : Avis clients vérifiés

### Phase 3 (Optimisation)
- **Contenu SEO** : Articles de blog, guides
- **Médias sociaux** : Intégration Instagram, Pinterest
- **Multilingue** : Version anglaise possible

## Gestion et mise à jour

### Workflow de création
1. **Brief créatif** : Définition des besoins par page
2. **Production** : Rédaction + photos selon guidelines
3. **Relecture** : Validation orthographe + cohérence
4. **Optimisation** : SEO + accessibilité
5. **Publication** : Intégration dans le CMS/site

### Fréquence de mise à jour
- **Portfolio** : Nouveau projet = nouvelle photo
- **Tarifs** : Révision annuelle
- **Contenu** : Mise à jour trimestrielle
- **Actualités** : Publication mensuelle (blog)

### Outils recommandés
- **Photos** : Appareil professionnel + éclairage studio
- **Édition** : Lightroom pour retouches, TinyPNG pour optimisation
- **Rédaction** : Google Docs pour collaboration
- **SEO** : Google Keyword Planner, SEMrush

## Critères qualité contenu

### Qualité textuelle
- **Orthographe** : Zéro faute (correction professionnelle)
- **Style** : Accessible, bienveillant, professionnel
- **Longueur** : Optimisée pour lecture web (50-70% du print)
- **Call-to-action** : Présents et variés

### Qualité visuelle
- **Résolution** : Minimum 1920px largeur pour web
- **Éclairage** : Naturel et flatteur
- **Composition** : Règles thirds, équilibre
- **Post-traitement** : Naturel, pas de surexposition

### Performance
- **Poids images** : < 200KB par image
- **Format** : WebP avec fallback JPG
- **Lazy loading** : Chargement progressif
- **CDN** : Distribution optimisée
